# Action Figure Release Tracker

This is a command-line action figure release tracker that can be used to manually track action figures that are being released.

## Requirements
* Python 3.10 or higher
* Windows

## Fields
Name: Name of the action figure

Release Date: Prospective date of release (Uses YYYY-MM-DD format)

Status: Leaked, Announced, Preorder, Released, Sold Out

Tier: Personal preference
    
* Preorder Now: Preorder straightaway
    
* Watchlist: Wait until 2 weeks before prospective release
    
* Release Day: Buy on release day
    
* Wait and Watch: Wait for reviews of the figure
    
* Pass: Safe to pass

Stores: Stores and their URLs

## How To Use
1. Download Python 3.10+ from [Python.org](https://www.python.org/)
2. Clone this repository.
3. Double-click on tracker.bat
